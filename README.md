# 🤖 AI Gesture Controller

An AI-powered desktop application that lets you control your computer using hand gestures through a webcam.

## ✨ Features

- 🖱️ Virtual Mouse
- ⌨️ Virtual Keyboard
- 🚀 Gesture Shortcuts
- 🔊 Volume Control
- 📷 Live Camera Preview
- 🎨 CustomTkinter GUI
- 🎬 Splash Screen

## 🛠 Technologies Used

- Python
- OpenCV
- MediaPipe
- CVZone
- CustomTkinter
- PyAutoGUI
- Pillow
- PyCAW

## 📂 Project Structure

```text
assets/
launcher.py
virtual_mouse.py
virtual_keyboard.py
gesture_shortcuts.py
splash.py
requirements.txt
```

## 🚀 Installation

```bash
git clone https://github.com/SaranyaLam/AI-Gesture-Controller.git
cd AI-Gesture-Controller
pip install -r requirements.txt
python splash.py
```
